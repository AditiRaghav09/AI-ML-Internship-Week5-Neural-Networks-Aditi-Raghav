# Introduction to Neural Networks and Deep Learning

## Overview

This project was completed as part of the AI/ML Internship Week 5 Task 1.

A simple Multi-Layer Perceptron (MLP) was developed using TensorFlow and Keras to classify the Iris dataset.

---

## Dataset

- Iris Dataset
- 150 Samples
- 4 Features
- 3 Classes

---

## Neural Network Architecture

- Input Layer: 4 neurons
- Hidden Layer 1: 8 neurons (ReLU)
- Hidden Layer 2: 4 neurons (ReLU)
- Output Layer: 3 neurons (Softmax)

---

## Hyperparameters

- Optimizer: Adam
- Loss Function: Categorical Crossentropy
- Epochs: 50
- Batch Size: 16
- Validation Split: 20%

---
